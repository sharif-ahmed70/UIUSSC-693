# UIUSSC - United International University Social Services Club

Project scaffold built with Next.js (App Router), TypeScript, and Tailwind CSS.

Setup

```bash
npm install
npm run dev
```

Files

- app/ - Next.js app routes and pages
- components/ - Reusable UI components
- data/ - Sample static data for events, notices, gallery, impact
- types/ - TypeScript types
- styles/ - global styles and Tailwind

Summary

This repo includes a responsive, semantic, and accessible UI for the UIUSSC website: Home, About, Events, Event details, Membership form UI, Gallery, Notices, and Contact.

Assumptions

- No backend or persistence; data lives in `data/` sample files.
- Branding uses navy/green direction; simple Tailwind theme tokens are included.
- Forms are UI-only and do not submit to a server yet.
